
# 📘 Кабинет партнёра DS Consulting
### Подробная инструкция для партнёра

*Простыми словами — как для человека, который впервые сел за компьютер.*
*Каждая страница разобрана по элементам: где что находится, как выглядит и что делает.*

## 🧭 С чего начать — основы для новичка

> Эти подсказки работают на **любой** странице кабинета.

| Понятие | Что это значит |
|---|---|
| 🖱 **Курсор** | Стрелка на экране, которая двигается вместе с мышкой. |
| 👆 **Нажать / кликнуть** | Навести курсор на нужное место и один раз коротко нажать **левую** кнопку мыши. |
| 📜 **Прокрутить** | Крутить колёсико в середине мыши: на себя — вниз, от себя — вверх. |
| ☰ **Меню** | Список разделов **слева** на экране. На телефоне прячется под значком из трёх полос. |
| ⬅️ **Назад** | Стрелка `←` в самом верху слева окна браузера. |
| ⌨️ **Поле для ввода** | Белый прямоугольник для текста: сначала кликните по нему, потом печатайте. |
| 👁 **Показать пароль** | Значок глаза в конце поля — превращает точки в буквы. |
| 🩶 **Серая кнопка** | Если кнопка бледная и не нажимается — заполнены не все поля. |
| ℹ️ **Значок «i»** | Наведите курсор — всплывёт подсказка с пояснением. |

# 1. 🔐 Вход и регистрация

## 🚪 Вход в кабинет

*Здесь вы вводите почту и пароль, чтобы попасть в кабинет.*

> 📍 **Как открыть:** страница появляется сама, когда вы заходите на сайт и ещё не вошли.

**👀 Что на экране**

- Слева (на компьютере) — зелёный блок с названием платформы. Только оформление.
- Справа — форма **«С возвращением»**:
  - 📧 поле **«Электронная почта»**;
  - 🔒 поле **«Пароль»** (со значком глаза справа);
  - 🟢 кнопка **«Войти в кабинет»**;
  - ссылка **«Забыли пароль?»**.

**🖱 Как войти**

1. Кликните по полю **«Электронная почта»** и наберите почту.
2. Кликните по полю **«Пароль»** и наберите пароль (при желании нажмите 👁, чтобы проверить).
3. Нажмите **«Войти в кабинет»** → откроется кабинет.

> ⚠️ Красная плашка «Неверная почта или пароль» — проверьте раскладку клавиатуры (рус/англ) и повторите.

**🔑 Если включён второй код (2FA)**

1. Откройте на телефоне приложение (Google Authenticator) — там 6-значный код, меняется каждые полминуты.
2. Наберите его в поле **«Код из приложения»**.
3. Нажмите **«Подтвердить»**.

## 📝 Регистрация нового кабинета

*Новый партнёр создаёт кабинет — **только по ссылке-приглашению** от наставника.*

> 📍 **Как открыть:** перейдите по ссылке-приглашению наставника. Без неё регистрация закрыта.

**🧩 Три шага сверху страницы:** `Ввод данных → Проверка → Готово`

**Шаг 1 — «Ввод данных»**

1. **«Фамилия» / «Имя» / «Отчество»** — русскими буквами.
2. **«Электронная почта»**, **«Телефон»**, **«Телеграм»** (`@ник`).
3. **«Дата рождения»** — через календарик.
4. **«Город»**.
5. **«Пароль»** и **«Подтверждение пароля»** — не короче 8 символов, минимум одна буква и одна цифра.
6. Поставьте ✅ галочку согласия на обработку данных.
7. Нажмите **«Далее →»**.

> ⚠️ Кнопка серая — значит какое-то обязательное поле пустое или телефон введён неверно.

**Шаг 2 — «Проверка»**

- Проверьте сводку данных (ФИО потом меняется только через поддержку).
- Прочитайте блок **«Стартовый период»**: за **90 дней** нужно набрать **≥ 500 баллов**, иначе партнёрство аннулируется.
- Нажмите **«Завершить регистрацию»** (или **«Назад»**, чтобы исправить).

**Шаг 3 — «Готово»**

- Зелёная галочка **«Регистрация завершена»**. Нажмите **«Перейти в кабинет»**.

## 🔁 Восстановление пароля

*Забыли пароль — попросите ссылку на почту.*

> 📍 **Как открыть:** на странице входа нажмите **«Забыли пароль?»**.

1. Кликните по полю **«Электронная почта»**, наберите почту.
2. Нажмите **«Отправить ссылку»**.
3. Откройте почту и перейдите по ссылке из письма (действует **60 минут**).

> ⏳ Повторно письмо — не чаще раза в 5 минут (кнопка покажет обратный отсчёт).

## 🔑 Установка нового пароля

> 📍 **Как открыть:** по ссылке из письма (почта уже подставлена).

1. Наберите **«Новый пароль»** (≥ 8 символов, буква + цифра).
2. Повторите его в поле **«Повторите пароль»**.
3. Нажмите **«Сохранить пароль»** → через пару секунд откроется страница входа.

# 2. 🧱 Как устроен кабинет: меню и верхняя полоса

*Слева — меню со всеми разделами. Сверху — узкая полоса (шапка) со значками.*

| | **Мои клиенты** | Список ваших клиентов |
| | **Контракты клиентов** | Договоры ваших клиентов |
| | **Контракты команды** | Договоры партнёров под вами |
| | **Структура** | Схема вашей команды |

### 🔝 Верхняя полоса (шапка, слева направо)

| Значок / элемент | Что делает |
|---|---|
| 🏷 Плашка статуса | Ваш статус («Активен») и дата конца периода; счётчик дней желтеет/краснеет к концу |
| 🟢 Индикатор системы | Ведёт на «Статус системы» |
| 🔗 **«Реф. ссылка»** | Копирует вашу ссылку-приглашение (только у активных) |
| 🌙 / ☀️ | Переключает тёмное / светлое оформление |
| 🔔 Колокольчик | Уведомления; красный кружок — есть новые; внутри — «Звук уведомлений» |
| 🟢 Кружок с инициалами | Меню пользователя: **«Профиль»** и **«Выйти»**; здесь же — сменить фото |

> 📱 На телефоне меню открывается значком ☰ вверху слева, а внизу — кнопки: Главная · Клиенты · Структура · Продукты · Профиль.

# 3. 🏠 Главная (Рабочий стол)

*Стартовая страница: главные цифры, контакты наставника, новости и быстрые кнопки.*

> 📍 **Как открыть:** пункт меню **«Главная»** (🏠).

**👀 Что на экране (сверху вниз)**

1. **Приветствие** — «Доброе утро/день/вечер, {имя}!» и дата. Только приветствие.
2. **«Мои показатели»** — плитки с цифрами:

   | Плитка | Что показывает |
   |---|---|
   | **ЛП** | Личные продажи (баллы за ваши сделки) |
   | **ГП** | Групповые продажи (баллы всей команды за месяц) |
   | **НГП** | Накопленные групповые продажи (определяют уровень) |
   | **Квалификация** | Ваш уровень и процент комиссии |
   | **Клиенты / Команда** | Сколько у вас клиентов и людей в команде |

3. 👑 **«Вы — Лидер сети»** — плашка, если вы во главе структуры.
4. **Карточки «Наставник» и «Лидер сети»** — ФИО, звание и кнопки-контакты:
   - 📞 телефон / 📧 почта → **копируются** при нажатии;
   - ✈️ Telegram → открывает чат.
5. 📰 **«Новости и объявления»** — сообщения от компании.
6. **«Активность команды»** — последние продажи команды (если есть).
7. **Правый столбец:** «Личные задачи» (список дел) · «Заметка» (блокнот) · «Сообщения» (🔴 New — непрочитанное) · «Ближайшие события».
8. ⚡ **«Быстрые действия»:** «Отчёт начислений» · «Калькулятор» · «Мои клиенты» · «Обратная связь» · «Профиль».

> 💡 Блок «Мои показатели» и часть кнопок видны только партнёрам-консультантам.

# 4. 📊 Обзор

## 📈 Дашборд (сводка за месяц)

*Наглядно: ваш уровень, прогресс до следующего, объёмы и команда за выбранный месяц.*

> 📍 **Как открыть:** пункт меню **«Дашборд»**.

**👀 Что на экране (сверху вниз)**

1. **Выбор месяца** (справа вверху) — нажмите → окошко выбора → все цифры обновятся.
2. 🟦 **«Активационный период»** (если проходите активацию) — дни, баллы, полоска прогресса.
3. **«Текущая квалификация»** — уровень, «Комиссия N%», кнопка **«Условия квалификаций»**, полоски **НГП** и **ОП по ГП**.
4. **«Объёмы»** — карточки (можно нажать): **ЛП** · **НГП** · **Объём 1 линии** (со сравнением к прошлому месяцу).
5. **«Отрыв»** — статус ветки:

   | Цвет | Значит |
   |---|---|
   | 🟢 Зелёный | «Отрыва нет» |
   | 🟡 Жёлтый | «Отрыв ≥ 70% — ветка не учитывается в ГП» |
   | 🔴 Красный | «Отрыв ≥ 90% — пул не выплачивается» |

6. **«Команда»** — партнёры 1 линии / всего / активные.
7. **«Клиенты»** — «Клиенты команды» и «Мои клиенты».
8. **«Партнёры по статусу»** — всего / зарегистрировано / активных / терминированных.

> 🖱 Кнопка **«Условия квалификаций»** открывает таблицу всех уровней (%, НГП, план по ГП, отрыв, пул). Нажатие на карточку «ЛП» или «НГП» открывает «Отчёт начислений».

## 💰 Отчёт начислений (ваши деньги за месяц)

*Подробно: сколько начислено за личные продажи, команду, бонусы, пул и итог «к выплате».*

> 📍 **Как открыть:** пункт меню **«Отчёт начислений»** (🏦).

**👀 Что на экране**

- Справа вверху — 🟢 **«Скачать XLSX»** (сохранить в Excel) и **выбор месяца**.
- Карточки: **Текущая квалификация** · **ЛП** · **ОП по ГП** · **НГП**.
- Карточки-итоги:

  | Карточка | Что внутри |
  |---|---|
  | **Личные продажи** | Баллы · бонус в баллах ЛП · бонус (₽) |
  | **Групповые продажи** | Баллы (ОП по ГП) · бонус по разнице % · бонус (₽) |
  | **Итого продажи** | Бонус (баллы) · бонус (₽) · пул (₽) · всего (₽) |
  | **Итог за месяц** | Баланс на начало · прочие начисления · итого · **К выплате** |

- Раскрывающиеся таблицы: **Личные продажи** · **Групповые продажи** · **Прочие начисления** · **Детали отрыва** · **Выплаты** (нажмите на заголовок, чтобы раскрыть).

> ⚠️ **Важно:**
> - Кнопка **«Скачать XLSX»** не показывается для месяцев **раньше июня 2026**.
> - Если вместо цифр плашка 🔒 **«Отчёт ещё не опубликован»** — данные готовятся, зайдите позже.
> - Прочерк «—» в столбце — поле для этого продукта не применяется.

## 🧾 Реестр выплат

*По месяцам: сколько начислено, выплачено и осталось получить.*

> 📍 **Как открыть:** пункт меню **«Реестр выплат»**.

**👀 Что на экране**

- **Переключатель месяца** — стрелки `‹` (прошлый) и `›` (следующий).
- Карточки-итоги: **Сальдо** · **Начислено** · **Пул** · **Прочее** · **К оплате** · **Остаток** (🟡 жёлтый — не всё выплачено, 🟢 зелёный — всё).
- 🏷 Плашка статуса: «Оплачено полностью / частично / в обработке».
- Таблица **«История выплат за месяц»**: Дата · Сумма · Статус · Комментарий.
- Таблица **«История по периодам»**: Период · Начислено · Пул · Прочее · Оплачено · Остаток · Статус.

> 💡 Главные цифры — **«К оплате»** и **«Остаток»**. Стрелка «вперёд» на текущем месяце не работает.

# 5. 💼 Работа

## 👥 Мои клиенты

*Список ваших клиентов и их данные. Только просмотр и поиск.*

> 📍 **Как открыть:** группа **«Работа»** → **«Мои клиенты»**.

**👀 Колонки таблицы**

| Колонка | Что в ней |
|---|---|
| ФИО клиента · Дата рождения · Место жительства | Основные данные |
| Телефон · Email | Контакты |
| Открытые продукты | Цветные плашки продуктов (нет — «—») |

**🖱 Поиск**

- Поля **«Поиск по ФИО…»**, **«Город»**, **«Email…»** — вводите, список обновляется сам.
- Кнопка **«Ещё»** → поиск по **дате рождения** (поля «с» / «по»).
- Кнопка **«Сбросить»** очищает поиск. Клик по заголовку **«ФИО клиента»** — сортировка.

---

## 📄 Контракты клиентов

*Все договоры, оформленные вашим клиентам: сумма, статус, баллы ГП и ваше вознаграждение.*

> 📍 **Как открыть:** группа **«Работа»** → **«Контракты клиентов»**.

**👀 Колонки таблицы**

| Колонка | Что в ней |
|---|---|
| Номер контракта · ФИО клиента | Кто и какой договор |
| Дата добавления · Дата открытия | Даты |
| Продукт · Программа · Срок контракта | Что оформлено |
| Сумма · Статус контракта | Деньги и состояние (цветная плашка) |
| **ГП (баллы)** · **Моё вознаграждение** | Ваши баллы и деньги за контракт |
| *(последний столбец)* | 🔘 Кнопка — открыть карточку контракта |

**🖱 Поиск и фильтры**

- Поля **«№ контракта»**, **«ФИО клиента»**; списки **«Продукт»**, **«Программа»** (активна после выбора продукта), **«Статус»**.
- Кнопка **«Ещё»** → диапазоны: Открыт · Заведён · Сумма · Срок, лет.

> ⚠️ Серый наклонный шрифт со значком **«~»** — **прогноз** (может измениться). Обычный чёрный — окончательное начисление.

---

## 🗂 Контракты команды

*Договоры всех партнёров под вами и ваше вознаграждение с них.*

> 📍 **Как открыть:** группа **«Работа»** → **«Контракты команды»**.

**👀 Отличия от «Контрактов клиентов»**

- Есть поле **«ФИО консультанта»** (партнёр команды) и общий поиск «ФИО клиента / № / продукт».
- Колонка **«ФИО ФК»** — зелёная фамилия, **на неё можно нажать**.

**🖱 Цепочка наставников**

1. Нажмите на **зелёную фамилию** в колонке **«ФИО ФК»**.
2. Откроется окно **«Ветка вверх»** — от консультанта через наставников до вас (вы — ⭐).
3. Закрыть — крестик в углу окна.

---

## 🌳 Структура

*Ваша команда в виде «дерева»: вы сверху, под вами ветками — партнёры.*

> 📍 **Как открыть:** группа **«Работа»** → **«Структура»**.

**👀 Колонки таблицы-дерева**

| Колонка | Что в ней |
|---|---|
| **Партнёр** | Имя (зелёное); слева ▸ треугольник — раскрыть команду |
| Уровень · Квалификация · Статус | Звание и состояние (плашки) |
| Дата смены статуса | + подпись «ЛП с активации: … / 500» у активных |
| **ЛП · ГП · НГП** | Баллы (можно сортировать) |
| Клиенты · Контракты · Партнёры | Счётчики |
| *(последний столбец)* | ⬇️ Скачать только эту ветку в Excel |

**🖱 Действия**

- **Раскрыть команду:** нажмите ▸ треугольник слева от имени (свернуть — ещё раз).
- **Карточка партнёра:** нажмите на зелёное имя → уровень, баллы, контакты, комментарии.
- **Комментарий:** в карточке поле **«Написать комментарий…»** → кнопка ✈️ (свой можно удалить).
- **Экспорт:** кнопка **«Экспорт структуры»** (вся) или ⬇️ в строке (одна ветка).

**🔍 Фильтры**

- **«ФИО партнёра…»**, **«Квалификация»** (галочками), **«Статус»** (Зарегистрирован / Активен / Терминирован / Исключён).
- **«Доп. фильтры»** → Дата рождения · Дата смены статуса · ЛП · ГП · НГП · Город.

> 💡 «ЛП с активации: … / 500» — сколько баллов новичку осталось до активации (🟡 не хватает, 🟢 выполнено).

# 6. 🎓 Развитие

## 📚 Обучение — главная

*Все ваши курсы. Отсюда начинают/продолжают учиться и заходят в Базу знаний.*

> 📍 **Как открыть:** пункт меню **«Обучение»**.

**👀 Что на экране**

- Счётчик: *«5 курсов · 2 в процессе · 1 ждут теста»* и поле **«Поиск по курсам»**.
- Полоса **«Продолжите с того места»** с кнопкой **«Продолжить»** (если курс начат).
- Карточка-портал **«База знаний»**.
- **Карточки курсов.** Метки на карточке:

  | Метка | Значит |
  |---|---|
  | 🟡 🔒 **«нужен тест»** | Тест по курсу ещё не сдан |
  | 🟢 🔓 **«открыто»** | Тест сдан, доступ к продукту получен |
  | ✅ **«изучен»** | Все уроки просмотрены (это ещё не «сдал тест») |

**🖱 Открыть курс:** наведите курсор на карточку и нажмите один раз.

---

## 📖 Страница курса

*Показывает, из чего состоит курс, и открывает уроки.*

- Вверху — **строчка-путь** (Обучение → Курс); слева — дерево уроков.
- **«Прогресс по курсу»** и карточка **«следующий урок»** (зелёная кнопка-стрелка).
- **«Структура курса»** — карточки разделов `M1`, `M2` с плашкой 🟢 «изучен» / 🟡 «в процессе», либо список уроков с кружком статуса.

**🖱 Открыть урок:** нажмите на строку урока или на карточку раздела.

---

## 📝 Урок

*Изучаете материал, отмечаете прохождение, сдаёте домашнее задание.*

**👀 Что на экране**

- Тонкая зелёная полоска вверху — сколько урока пролистано.
- Справа вверху — кнопка **«Урок изучен»** / плашка **«Изучено»** / 🔒 **«Закрыт»**.
- Содержимое: текст, видео, блок **«Материалы и ссылки»**.
- Внизу — кнопки **«Предыдущий» · «К модулю» · «Следующий»** (или **«К тесту»**).

**🖱 Отметить урок изученным**

1. Прочитайте/посмотрите материал (прокрутите донизу).
2. Нажмите зелёную кнопку **«Урок изучен»** → появится плашка **«Изучено»** и конфетти.

> 💡 Можно не нажимать отдельно — кнопка **«Следующий»** отметит урок сама.

**📎 Домашнее задание**

1. Наберите ответ в поле **«Ваш ответ»**.
2. Прикрепите файл кнопкой **«Прикрепить файл»** (скрепка) → выберите файл → «Открыть».
3. Нажмите **«Отправить на проверку»** → появится **«⌛ На проверке»**.
4. Ждите: 🟢 **«✓ Принято куратором»** или 🔴 **«✗ Отклонено — попробуйте ещё раз»**.

> ⚠️ Если кнопка «Урок изучен» бледная — сначала нужно сдать домашнее задание.

## ✅ Итоговый тест курса

*Сдаёте тест — курс пройден, открывается допуск к продаже продукта.*

> 📍 **Как открыть:** кнопка **«Пройти тест»** на уроке-тесте или **«К тесту»** на последнем уроке.

**Правило (вверху справа):** `попыток без ограничения · для допуска нужно 100%`.

**🖱 Как проходить**

1. Прочитайте вопрос, нажмите на нужный вариант (рамка выделится).
2. Нажмите **«Далее»** → следующий вопрос. (Пока ответ не выбран — «Далее» бледная.)
3. Между вопросами — кнопки **«Назад»/«Далее»** или номера-кружочки внизу.
4. На последнем — **«Отправить ответы»**.

| Результат | Что увидите |
|---|---|
| 🟢 **Пройден** | Галочка, конфетти, кнопка **«Перейти к продукту»**, ссылка «в Обучение» |
| 🔴 **Не пройден** | Красная плашка, номера ошибок, кнопки **«Пройти ещё раз»** / «к материалам урока» |

# 8. 👤 Профиль

*Личные данные, документы, реквизиты для выплат, защита входа и смена пароля.*

> 📍 **Как открыть:** кружок с инициалами (правый верх) → **«Профиль»**.

Слева — список разделов. Разберём каждый.

### 🧍 Личные данные
ФИО (только чтение, меняется через поддержку), дата рождения, пол, страна, город, email, телефон, Telegram.
→ измените поле → **«Сохранить»** (или **«Отменить»**).

### 🪪 Документы
**«Паспорт (разворот с фото)»** и **«Паспорт (регистрация)»** → выберите файл → **«Загрузить»**.

### 🏦 Реквизиты (данные для выплат)

> Реквизиты проверяет сотрудник вручную — это «верификация».

1. Нажмите на поле **«ИНН ИП»**, наберите ИНН.
2. Уберите курсор / нажмите **Enter** — система заполнит **Наименование**, **ОГРНИП**, **Юр. адрес**.
3. Галочка **«Адрес регистрации совпадает с фактическим»** (снимите — заполните **«Фактический адрес»**).
4. Заполните **«Email для документов»**, **«Телефон ИП»** → **«Сохранить»**.
5. Блок **«Банковские реквизиты»**: банк, БИК, расчётный счёт, корр. счёт, получатель → **«Сохранить»**.

| Метка | Значит |
|---|---|
| 🟡 На проверке | Реквизиты отправлены сотруднику |
| 🟢 Подтверждено | Проверка пройдена |
| 🔴 Отклонено | С причиной — исправьте и сохраните снова |

> ⚠️ Реквизиты можно менять, но не удалять. Подтверждённый банк меняется через **«Сменить реквизиты» → «Отправить на проверку»** (выплаты временно приостановят).

### 🔐 Безопасность — второй код (2FA) и пароль

**2FA** — дополнительный одноразовый код при входе (пароль + 6 цифр из приложения на телефоне).

1. **«Безопасность» → «Включить 2FA»** → появится QR-код.
2. В приложении (Google Authenticator) отсканируйте QR-код (или введите **«Секрет»** вручную).
3. Наберите 6-значный код в поле → **«Подтвердить и включить»**.

**Смена пароля:** введите **«Текущий пароль»**, **«Новый пароль»**, **«Подтверждение»** → **«Сменить пароль»**.

### 💬 Telegram-bot
**«Привязать через бота» → «Открыть в Telegram» → Start** (15 минут). Статус станет **«Подключён»**.

### 🔗 Реферальные ссылки
**«Реферальный код»** и **«Ссылка для приглашения»** со значком копирования (у активных).

# 9. 🔒 Ограниченный доступ

*Эту страницу видит партнёр со статусом **«Терминирован»** — доступ к разделам закрыт.*

- 🔒 Заголовок **«Доступ ограничен»**.
- **«Что это значит»:** условия стартового периода не выполнены; баллы обнулены, клиенты и договоры переданы вышестоящему; повторная регистрация — не более 3 раз.
- **«Для восстановления доступа»:** связаться с поддержкой, обсудить план с наставником.
- Кнопки: **«Обратная связь»** и **«Профиль»** (только они и работают).

## 📖 Словарь терминов

| Термин | Простыми словами |
|---|---|
| **ЛП** | Личные продажи — баллы за ваши личные сделки |
| **ЛО** | Личный объём для старта — нужно 500 баллов за 90 дней |
| **ГП** | Групповые продажи — ваши баллы + баллы всей команды |
| **НГП** | Накопленный групповой объём — по нему растёт уровень |
| **ОП по ГП** | Обязательный план за месяц; не выполнить — часть денег удержат |
| **Квалификация** | Ваш уровень (1–10) с фиксированным процентом комиссии |
| **Отрыв** | Одна ветка даёт слишком много ГП: 70% — не в счёт, 90% — нет пула |
| **Пул** | Доп. выплата для старших уровней (6+) при выполнении условий |
| **Прогноз** | Ещё не окончательное число (серый курсив со значком «~») |
| **ИП на УСН** | Форма предпринимателя, в которой работает партнёр |
| **Верификация** | Ручная проверка ваших реквизитов перед выплатами |
| **2FA** | Двойная защита входа: пароль + код из приложения |
