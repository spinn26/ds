**Бизнес-смысл:** Анализ карьерного роста и эффективности структуры. Позволяет увидеть динамику продаж (ЛП/ГП) каждого партнера в сравнении с прошлым месяцем.

**Тип отчёта:** Сравнительная таблица (Прошлый период vs Текущий период).

| **Название поля** | **Тип данных** | **Пример значения** | **Описание логики** |
|----|----|----|----|
| **Партнёр / Активность** | Строка | ```javascript
Азамова Яна
```, ```javascript
Активный
``` | Базовые данные партнера. |
| **Статус** | Строка | ```javascript
Финансовый консультант
``` | Бизнес-роль на платформе. |
| **Пред Квалификация** | Строка | ```javascript
3 [Expert]
``` | Единый статус в прошлом месяце. |
| **Пред ЛП / ГП / НГП** | Число | ```javascript
138.293845
``` | Объемы прошлого месяца (точность 6 знаков). |
| **Текущ Квалификация** | Строка | ```javascript
4 [FC]
``` | Единый статус в текущем (выбранном) месяце. |
| **Текущ ЛП / ГП / НГП** | Число | ```javascript
450.000000
``` | Объемы текущего месяца. |
