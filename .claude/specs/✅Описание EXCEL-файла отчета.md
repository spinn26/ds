##  2. Лист «Личные продажи»

**Назначение:** детализация каждой личной продажи (сделки, где сотрудник — непосредственно продавец).

**Состав полей (колонки):**

| Колонка | Описание |
|----|----|
| Дата | Дата сделки |
| Контракт | Номер контракта |
| Клиент | ФИО клиента |
| Продукт | Название продукта |
| Программа | Программа / тариф |
| Сумма оплаты | Сумма в рублях с копейками |
| Сумма без НДС | Сумма без НДС |
| Параметр | Тип начисления (Стандарт / МФ / Апфронт и др.) |
| ЛП | Баллы за личные продажи |
| Бонус | Бонус в баллах |
| Бонус, ₽ | Бонус в рублях |
| Комментарий | Дополнительная информация (удержания, коррекции, комиссии) |

##  3. Лист «Групповые продажи»

**Назначение:** детализация продаж, совершённых партнёрами / подчинёнными (групповой объём).

**Состав полей (колонки):**

| Колонка | Описание |
|----|----|
| Дата | Дата сделки |
| Контракт | Номер контракта |
| Клиент | ФИО клиента |
| Партнёр сделки | Кто фактически совершил продажу (партнёр, подчинённый) |
| Продукт | Название продукта |
| Программа | Программа / тариф |
| Сумма оплаты | Сумма в рублях |
| Сумма без НДС | Сумма без НДС |
| Параметр | Тип начисления (Стандарт / Апфронт / МФ) |
| ГП | Баллы за групповые продажи |
| Бонус | Бонус в баллах (обычно 15% от ГП) |
| Бонус, ₽ | Бонус в рублях |
| Комментарий | Дополнительная информация (комиссии, удержания, пометки о компрессии) |

## 4. Лист «Прочие начисления и выплаты»

**Назначение:** корректировки, удержания, дополнительные начисления (не связанные напрямую с продажами).

**Состав полей (колонки):**

| Колонка | Описание |
|----|----|
| Дата | Дата операции |
| Сумма оплаты | Сумма в рублях (может быть отрицательной — удержание) |
| Баллы | Сумма в баллах (может быть отрицательной) |
| Комментарий | Описание: удержание за конкретный контракт / разовое начисление (например, «15% до уровня Co-founder») |
