**Бизнес-смысл:** Кадровый и аудиторский отчёт. Показывает список партнеров, у которых изменился статус активности (например, они были отключены от платформы из-за невыполнения условий).

**Тип отчёта:** Линейный список пользователей.

| **Название поля** | **Тип данных** | **Пример значения** | **Описание логики** |
|----|----|----|----|
| **ФИО** | Строка | ```javascript
Харченко Анна Евгеньевна
``` | Полное имя партнера. |
| **Статус** | Строка | ```javascript
Терминирован
``` | Текущий или присвоенный статус в указанную дату. |
| **Фактическая дата терминации/смены статуса** | Timestamp | ```javascript
2026-02-03
``` | Точное время по UTC, когда система зафиксировала изменение статуса (сработал скрипт). |
| **Плановая дата терминации**  | Дата | ```javascript
2026-02-02
``` | Плановая дата отключения (обычно на 1 день раньше технической детерминации). |
