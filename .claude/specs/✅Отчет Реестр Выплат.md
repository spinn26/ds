**Бизнес-смысл:** Мастер-отчёт для бухгалтерии. Является основой для формирования реальных банковских платежек. Содержит кредиторскую задолженность компании перед партнерами.

**Тип отчёта:** Детализация баланса и реквизитов.

| **Название поля** | **Тип данных** | **Пример значения** | **Описание логики** |
|----|----|----|----|
| **ФИО** | Строка | ```javascript
Абазов Алексей Хашаович
``` | Имя получателя платежа. |
| **Активность** | Строка | ```javascript
Активный
``` / ```javascript
Терминирован
``` | Текущее состояние профиля. |
| **Сальдо** | Число | ```javascript
2250.00
``` или ```javascript
-11692.72
``` | Входящий остаток с прошлых периодов (минус = долг партнера). |
| **Начислено** | Число | ```javascript
35000.00
``` | Сумма комиссий по сделкам за месяц. |
| **Прочие / Пул** | Число | ```javascript
0.00
``` или ```javascript
5000.00
``` | Ручные корректировки и лидерские бонусы. |
| **Итого начислено** | Число | ```javascript
40000.00
``` | Начислено + Прочие + Пул. |
| **Итого к оплате** | Число | ```javascript
42250.00
``` | Сальдо + Итого начислено. |
| **Оплачено** | Число | ```javascript
0.00
``` | Фактически выплаченная сумма. |

320...

true

false
``` | Прошел ли профиль модерацию. |
