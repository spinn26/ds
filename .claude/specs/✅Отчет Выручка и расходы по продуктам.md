**Бизнес-смысл:** Агрегированный (сводный) финансовый срез. Нужен руководству для быстрого анализа рентабельности и прибыльности конкретных продуктов без погружения в детали каждой сделки.

**Тип отчёта:** Группировка сумм по продуктам (одна строка = один продукт).

| **Название поля** | **Тип данных** | **Пример значения** | **Описание логики** |
|----|----|----|----|
| **Продукт** | Строка | ```javascript
Axevil
``` или ```javascript
Образовательные продукты
``` | Название продукта. Ключ для группировки данных. |
| **Доход** | Число | ```javascript
1292266.03
``` | Суммарный доход компании по всем транзакциям этого продукта за период. |
| **Расход** | Число | ```javascript
639979.37
``` | Суммарный расход (или себестоимость) по всем транзакциям этого продукта. |
