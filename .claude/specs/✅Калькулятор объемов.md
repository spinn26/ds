## 1. Назначение инструмента

Калькулятор предназначен для моментального расчёта Личного объёма (ЛП) и комиссионного вознаграждения партнёра (в баллах и рублях) на основе параметров конкретной сделки.

Используем данные из таблицы (которую заливаем в БД если эти данные отсутсвуют в новой модели) - <https://docs.google.com/spreadsheets/d/1ysPw8zSESzs18zlvGEYHlW1sTYyvvZ08thEf6BYauEY/edit?gid=531018859#gid=531018859> - заливаем всю таблицу (если этих данных нет в новой БД), она так же используется для рассчета коммисионных, столбцы - продукт, програма, валюта, поставщик, %ДС, свойство, срок контракта, годы выплаты КВ, последний столбец - категория продуктов - нужен для фильтра в разделе для партнеров. 

В калькулятор выводим только продукты которые имеют заливку зеленого цвета. Остальные нужны для рассчета по старым контрактам но мы их не продаем, желтые еще под вопросом продукты. 

также в калькуляторе используется таблица квалификация которую берем из документа <https://docs.google.com/document/d/1qH_16jGPJZK8L6euObjslpJMJy0OcHfk_5iuVkkblvY/edit?tab=t.0>


Если продукт не рублевый для пересчета используем курс с раздела “Валюты и НДС” за предыдущие период (месяц)

## 4. Логика функционала расчета

### 1. Данные из Базы Данных (Справочные значения)

На основе параметров, выбранных пользователем (Продукт + Программа + Срок), система фильтрует базу данных и находит **одну целевую строку**. Из этой строки система забирает следующие значения для расчёта:

- **Валюта** (RUB, USD, EUR).
- **% DS** — процент отчисления (например: 25,00%, 77,50%, 3,45%). *В коде этот процент должен конвертироваться в десятичную дробь (например, 77,50% = 0,775).*
- **Ставка квалификации** — процент вознаграждения, соответствующий квалификации партнёра, тоже конвертируется в десятичную дробь (Start = 15%, Pro = 20%, Expert = 25% и т.д.).

---

### 2. Универсальный алгоритм расчёта (Шаги для кода)


**Шаг 1. Приведение суммы к базовой валюте (Рубли)** Система проверяет значение переменной «Валюта» из найденной строки БД:

- Если валюта **RUB**: Базовая сумма = Сумма оплаты от клиента
- Если валюта **не RUB**: Базовая сумма = Сумма оплаты от клиента * Курс валюты (за предыдущий период) 


**Шаг 2. Расчёт чистой выручки компании (очистка от НДС)** Система применяет комиссию продукта (% DS из базы) к Базовой сумме и очищает её по формуле:

- **Формула Выручки:** Выручка DS = (Базовая сумма * % DS) / 105 * 100


**Шаг 3. Расчёт Личного Объёма (ЛП в баллах)** Для перевода выручки в баллы объёма (согласно правилу 1 балл = 100 руб.), система делит полученную выручку на 100.

- **Формула ЛП:** ЛП = Выручка DS / 100 *(Примечание для кодера: математически Шаг 2 и Шаг 3 можно сократить до ЛП = Базовая сумма * % DS / 105, но для прозрачности архитектуры лучше считать пошагово).*


**Шаг 4. Расчёт комиссии партнёра в баллах** Система берет рассчитанный объём ЛП и умножает его на процент квалификации партнёра.

- **Формула:** Комиссия (баллы) = ЛП * Ставка квалификации


**Шаг 5. Расчёт итоговой комиссии в рублях** Согласно маркетинг-плану, стоимость 1 балла на выплату фиксирована и равна 100 рублям.

- **Формула:** Комиссия (рубли) = Комиссия (баллы) * 100

### *3. Примеры (Test Cases) для проверки логики разработчиком*


**Пример 1: Валютный продукт со сложной фильтрацией (Investor Trust Evolution)**

- **Ввод пользователя:** Продукт = Investor Trust Evolution, Программа = EVO, Срок контракта /Год выплаты КВ= 25 / 1 год, Сумма = 500, Курс = 95, Квалификация = Expert (ставка 25% или 0,25).
- **Данные из БД:** Валюта = USD, % DS = 77,50% (0,775).
- **Расчёт системы:**
  1. Базовая сумма: 500 * 95 = **47 500 руб.**
  2. Выручка DS: (47 500 * 0,775) / 105 * 100 = **35 059,52 руб.**
  3. ЛП: 35 059,52 / 100 = **350,60 баллов**.
  4. Комиссия (баллы): 350,60 * 0,25 = **87,65 балла**.
  5. Комиссия (рубли): 87,65 * 100 = **8 765 рублей**.


**Пример 2: Простая рублёвая услуга (ФНС Налоговое консультирование)**

- **Ввод пользователя:** Продукт = ФНС, Сумма = 20 000, Квалификация = Pro (ставка 20% или 0,20).
- **Данные из БД:** Валюта = RUB, % DS = 25,00% (0,25).
- **Расчёт системы:**
  1. Базовая сумма: **20 000 руб.** (умножение на курс не требуется).
  2. Выручка DS: (20 000 * 0,25) / 105 * 100 = **4 761,90 руб.**
  3. ЛП: 4 761,90 / 100 = **47,62 балла**.
  4. Комиссия (баллы): 47,62 * 0,20 = **9,52 балла**.
  5. Комиссия (рубли): 9,52 * 100 = **952 рубля**.


**Пример 3: Из документа с Маркетинг-планом (для проверки совпадения)**

- **Условия из твоего документа:** Эволюция, Взнос 100 000 руб, % DS = 87% (0,87), Квалификация Start (15%).
- **Расчёт системы:**
  1. Базовая сумма: **100 000 руб.**
  2. Выручка DS: (100 000 * 0,87) / 105 * 100 = **82 857,14 руб.** 
  3. ЛП: 82 857,14 / 100 = **828,57 баллов**.
  4. Комиссия (баллы): 828,57 * 0,15 = **124,29 балла**.
  5. Комиссия (рубли): 124,29 * 100 = **12 429 рублей**.
